# 📊 CX Report Automation — MELI

Sistema de automação para geração de relatórios executivos de Customer Experience (CX), integrando coleta de dados, processamento e entrega automatizada.

## 🚀 Objetivo
Automatizar o processo de criação de relatórios mensais, reduzindo esforço manual e aumentando a eficiência operacional.

## ⚙️ Tecnologias
- Python
- Microsoft Forms
- Power Automate
- Excel (SharePoint)
- Power BI

## 🧠 Funcionalidades
- Coleta automatizada de dados via Microsoft Forms
- Integração com Excel hospedado no SharePoint
- Processamento e estruturação de dados
- Geração de relatórios executivos automatizados
- Fluxos automatizados com Power Automate

## 📊 Impacto
- Redução significativa de esforço manual
- Padronização dos relatórios
- Maior velocidade na entrega de materiais executivos

## 🔄 Fluxo do Sistema
Forms → Excel (SharePoint) → Processamento → Relatório → Envio automático

## ⚠️ Observação
Este projeto é uma adaptação para portfólio baseada em um caso real, sem exposição de dados sensíveis.
